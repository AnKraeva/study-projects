# 📞 Telemarketing Sales Analysis

Подготовка и очистка данных телемаркетинга для оценки эффективности продаж.

## Цель

Очистить и объединить данные о продажах и подключениях продуктов, чтобы выделить только корректные сделки.

---

## Данные

	* `tm_sales_1/2/3` — данные о продажах
	* `prod_activations_logs` — логи подключений

Особенности:

	* пропуски в `SUBS_ID`
	* неполные идентификаторы
	* разный порядок колонок

---

## 🔍 Что сделано

	* предобработка и очистка данных
	* нормализация идентификаторов пользователей
	* объединение данных из разных источников
	* фильтрация продаж по правилу: подключение считается валидным, если длится ≥ 5 минут

---

## Результат

Получен очищенный датасет корректных продаж, пригодный для дальнейшего анализа эффективности телемаркетинга.
