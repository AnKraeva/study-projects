# 🧪 A/B Testing Analysis

## Цель

Оценить влияние новой механики оплаты на ключевые продуктовые метрики.

## Данные

Данные эксперимента:

* контрольная и тестовая группы
* активность пользователей
* данные об оплатах

## Что сделано

* предобработка данных;
* расчет метрик:
  * CR (конверсия в оплату),
  * ARPPU,
  * ARPAU;
* сравнение групп;
* проверка статистической значимости.

## Результат

Обнаружено статистически значимое увеличение средней выручки на пользователя в тестовой группе → новую механику можно внедрять.

## Tools

Python, pandas, numpy, matplotlib, seaborn, статистические тесты
